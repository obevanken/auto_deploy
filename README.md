# auto_deploy
  Deploy for GitLab

# Application
  
Create **labhook.json** file in project root :
  
 ``` 
 {
    "repo_name": "Repository name",
    "path": "path to the project",
    "branch": "Name branch",
    "commands": [
        "youre command 1",
        "youre command 2"
    ]
} 
```
install requirements: ```pip install -r requirements.txt```
